# [UltraStaff](https://www.spigotmc.org/resources/ultrastaff-1-16-1-21-staff-moderation-plugin.108913/)

**UltraStaff** is a **Minecraft** plugin designed specifically for server staff, providing essential moderation tools and features for any server environment.

---

## ‚ú® Features
- **Multiversion Support:** 1.16 - 1.20  
- **StaffMode:** Easily toggle staff mode  
- **Vanish:** Enable or disable invisibility  
- **Fly Mode:** For staff convenience  
- **InventorySee:** View player inventories  
- **Random Teleport:** Teleport to random players  
- **Freeze:** Temporarily freeze players  
- **Navigator:** Quick teleportation and server navigation  
- **Customizable Messages & Items**  
- **StaffChat:** Private chat for staff members  
- **Discord Integration**  
- **Report System (BETA)**  
- **Chat Management Module**  
- **RGB Support for Messages**  
- **More Features Coming Soon‚Ä¶**

## üíª Commands
| Command | Description |
|---------|-------------|
| `/staffmode` | Enable or disable staff mode |
| `/vanish` | Enable or disable vanish mode |
| `/freeze <player>` | Freeze a player |
| `/fly [player]` | Toggle fly mode |
| `/s <player>` | Teleport a player to your location |
| `/invsee <player>` | Open a player's inventory |
| `/staffchat [message]` | Send a message in staff chat |
| `/chat [clear/mute]` | Mute or clear chat |
| `/report` | Report a player |
| `/reports <player/discard>` | Manage or discard reports |
